# SpaceWar
A simple game on python

 ### How to run: ###
 1. open terminal
 2. go to game directory(`cd name/`)
 3. write `python app.py`
 
  ### How to play ###
  
  Use <kbd>→</kbd> to go right and <kbd>←</kbd> to go left as simple as that also use <kbd>Space</kbd> to shoot
